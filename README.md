# FinTrack

FinTrack is a personal finance management system built for learning purposes and portfolio development using the .NET ecosystem.

## Project Goals

The project aims to help users manage income, expenses, and financial categories while providing intelligent spending insights through AI integration.

## Tech Stack

### Backend

* C#
* ASP.NET Core Web API
* Entity Framework Core
* PostgreSQL

### Mobile

* .NET MAUI
* CommunityToolkit.Mvvm
* Refit

### Quality & Testing

* FluentValidation
* Serilog
* xUnit
* Moq

### Artificial Intelligence

* Google Gemini API
